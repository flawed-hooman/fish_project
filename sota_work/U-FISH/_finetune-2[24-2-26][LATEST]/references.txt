Reads YOLO format labels (class xc yc w h, normalized 0-1) and extracts ONLY the center coordinates (xc, yc), converting them to pixel coordinates in (row, col) format. The bounding box width/height are completely discarded for training — U-FISH is a point detection model, not a box detection model.

Step 3: Target Density Map — make_density_map_official()
This is the core function that generates the ground truth the model learns to predict. It replicates the exact method from the official U-FISH source code (ufish/data.py):

Create a binary point mask: zeros everywhere, 1.0 at each spot center coordinate
Apply scipy.ndimage.gaussian_filter(mask, sigma=1.0) — each point becomes a tiny ~3px-wide Gaussian blob
Peak-normalize: read the filtered value at each spot center, divide the entire density map by the minimum peak value → ensures every spot peak reaches ≈ 1.0
Why σ=1 matters: With σ=1, each spot becomes a tight ~3px peak. The U-FISH local_maxima post-processing can cleanly find exactly one peak per spot. Your previous approach used σ=3-8 (calculated from YOLO box sizes), which created 15-25px blobs that the post-processing fragmented into multiple false detections per spot.

Step 4: Input Preprocessing — load_and_preprocess()
This is where your RGB images are converted to single-channel inputs:

Load RGB image as uint8 → convert to float32 [0, 1]
Detect probe type from filename (get_probe_channel()): ORANGE→Red(0), FITC→Green(1), AQUA→Blue(2)
Extract probe channel — take only the relevant RGB channel
DAPI subtraction (if SUBTRACT_DAPI=True AND probe is NOT AQUA): subtract the Blue channel from the probe channel, clip to [0, 1]. This removes DAPI nuclear staining bleed-through that would otherwise confuse the model
Scale to [0, 255] and apply scale_image() from ufish.utils.img — this does 99.99th percentile outlier clipping + skimage.exposure.rescale_intensity to map the data to float32 [0, 255]. This is the exact function the pretrained U-FISH model was trained with
Final input tensor: shape (1, 512, 512), dtype float32, range [0, 255]
